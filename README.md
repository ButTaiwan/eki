# 站名牌產生器 / 駅名標ジェネレータ

> 原本 PHP 版工具的純前端重製版本。  
> 現在可直接用瀏覽器操作（含中文 / 日文頁面）。

---

## 中文說明（繁體）

### 這是什麼
這是一個可產生各地鐵道 / 捷運「站名牌風格」圖像的工具。  
你可以輸入站名、前後站資訊、站號、路線色等資料，快速輸出 PNG 圖檔。

### 主要功能
- 可模擬多種風格站名牌（台灣、日本、韓國、中國、英國等）
- 預覽畫布輸出品質提升，可下載高解析 PNG
- 可選擇不同下載尺寸（XL / X / M / S / XS）
- 支援中、日文兩個操作頁面（`index.html` / `ja.html`）
- 可從資料庫快速帶入系統 / 路線 / 車站資訊
- 中文繁簡轉換功能（產生新加坡、中國站名牌時可用）

### 使用方式
1. 開啟 `index.html`（中文版）或 `ja.html`（日文版）
2. 選擇樣式、系統、路線與車站
3. 視需要編輯站名與前後站資料
4. 點擊「繪製」並下載 PNG

### 提醒與免責事項
- 本工具以趣味與創作為主，不以重現真實各站站名標細節為目的。
- 首次使用需下載多個字型檔（單檔常達十幾 MB），請留意網路環境與讀取時間。
- 字型授權整理於 `assets/fonts/License.txt`，若有疑慮請在 Issues 提出。

## 日本語説明

### このツールについて
このプロジェクトは、駅名標風デザインを画像として生成できるツールです。  
駅名・前後駅情報・駅番号・路線カラーなどを入力し、PNG 画像として保存できます。

### 主な機能
- 多様な駅名標スタイルを再現（台湾・日本・韓国・中国・英国など）
- プレビュー / 出力品質を改善し、高解像度 PNG を保存可能
- ダウンロードサイズを選択可能（XL / X / M / S / XS）
- 中国語版 / 日本語版の 2 ページを用意（`index.html` / `ja.html`）
- データからシステム / 路線 / 駅を読み込み、入力を素早く補完
- 繁体字↔簡体字変換機能を搭載（シンガポールと中国の駅名標向け）

### 使い方
1. `index.html`（中国語）または `ja.html`（日本語）を開く
2. スタイル・システム・路線・駅を選ぶ
3. 必要に応じて駅名や前後駅情報を編集する
4. 描画して PNG を保存する

### 注意事項・免責
- 本ツールは主に趣味用途です。実在駅名標の細部再現を保証するものではありません。
- 初回利用時は大型フォントを複数ダウンロードします（1 ファイル十数 MB 規模）。通信環境にご注意ください。
- フォントライセンスは `assets/fonts/License.txt` にまとめています。懸念点があれば Issues でご連絡ください。
